# Rural Timber Boilerplate Theme

This is the theme inside the `Rural Timber Boilerplate` WordPress repository.

For the full setup guide and the complete getting-started steps, start with the root [README](../../../README.md).

## Theme Folder

```text
wp-content/themes/weerts
```

## Commands

Run these from the theme directory:

```bash
composer install
pnpm install
pnpm run dev
pnpm run build
```

## Main Theme Files

- `functions.php`: theme bootstrap and asset loading
- `templates`: Twig templates
- `templates/partials`: shared Twig partials
- `css/main.css`: front-end Tailwind entry
- `css/editor.css`: editor Tailwind entry
- `js/index.js`: JavaScript entry
- `static`: compiled assets
